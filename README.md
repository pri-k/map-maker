# Map Maker
<p>A simple Python program made using Folium and Geopy to generate a map of any city in the world.<br>
Depending on the user input the program generates an HTML file containing the map of the city.</p>

## Requirements
Folium
```
pip install folium
```
Geopy
```
pip install geopy
```
